# 音效资源说明

本目录用于存放游戏所需的音效文件。

## 建议的音效文件

1. **bgm.mp3** - 背景音乐
2. **deal.mp3** - 发牌音效
3. **play.mp3** - 出牌音效
4. **win.mp3** - 胜利音效
5. **lose.mp3** - 失败音效
6. **bomb.mp3** - 炸弹音效
7. **click.mp3** - 点击音效

## 临时方案

目前音效功能未完全实现，游戏可以静音运行。

## 音效来源建议

- freesound.org（免费音效库）
- zapsplat.com（免费音效资源）
- 自己录制或制作
- 使用AI生成音效

## 注意事项

- 音效文件建议使用MP3格式（兼容性好）
- 单个文件大小建议不超过1MB
- 背景音乐可以适当长一些（循环播放）
- 音效应该简短明快
