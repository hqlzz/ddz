# 图片资源说明

本目录用于存放游戏所需的图片资源。

## 需要的图片文件

1. **avatar-player.png** - 玩家头像（建议尺寸：100x100px）
2. **avatar-farmer.png** - 农民头像（建议尺寸：100x100px）
3. **avatar-landlord.png** - 地主头像（可选，建议尺寸：100x100px）

## 临时方案

目前游戏使用纯CSS渲染，不需要图片也可以正常运行。

如果需要添加头像图片，可以：
1. 准备相应的PNG图片
2. 放置在此目录下
3. 确保文件名与上述列表一致

## 图片来源建议

- 可以使用免费图标网站（如：iconfinder.com、flaticon.com）
- 自己绘制或设计
- 使用AI生成头像
